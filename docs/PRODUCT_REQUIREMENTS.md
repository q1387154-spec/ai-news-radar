# 物流快递行业 AI 政策情报雷达
## 产品需求文档 · 中通吉专属版

> 基于 AI News Radar（伯乐Skill）fork 定制
> 目标用户：上海中通吉网络技术有限公司 · 申报/人事/技术团队
> 优先级：青浦/长三角示范区 → 上海市 → 国家级

## 一、产品定位

| 维度 | 原版 AI News Radar | 中通吉专属版 |
|------|-------------------|-------------|
| 目标用户 | 普通 AI 爱好者 | 物流/快递企业申报人员、人事HR、技术人员 |

| 地域聚焦 | 全球化 AI 新闻 | **青浦/长三角 → 上海 → 国家** 三级穿透 |

**一句话定位**：帮中通吉团队每天扫一遍"有没有跟我物流业务相关的新政策、新补贴、新技术、新机会"。

## 三、核心功能模块

### 3.1 保留（原版功能，直接复用）

| 模块 | 复用说明 |
|------|---------|
| 24小时 AI 新闻流 | 复用原版抓取/去重/过滤 pipeline |

### 3.2 改造（原版基础上定制）

**① 首页默认视图**
- 默认 Tab：`政策申报`（原版默认是"全部"）
- 新增固定 Banner：`申报窗口期倒计时`（如"高企认定 - 预计6月截止"）

**② 分类标签体系（替换原版标签）**

| 标签 | 对应内容 |
|------|---------|
| `🏛 政策申报` | 高企、科小、加计扣除、职业技能补贴、稳岗 |
| `🚚 物流科技` | 无人配送、无人机、低空经济、智慧物流 |
| `🤖 AI技术` | 大模型、物流场景应用、技术工具 |
| `💼 招聘求职` | 事业单位、央企、社招公告 |
| `💰 财税补贴` | 税收优惠、超长期国债、设备更新补贴 |
| `📋 监管合规` | 法规、条例、备案要求 |

**③ 筛选逻辑**
- 地域维度：青浦 / 长三角 / 上海 / 国家
- 资金维度：市级补贴 / 国家级补贴 / 税收减免
- 截止维度：即将截止（7天内）/ 本月 / 已过

### 3.3 新增（需开发）

**① 申报窗口期日历**
- 入口：顶部 Tab `申报日历`
- 内容：当年重要申报节点（高企认定、科小入库、职业技能补贴申请等）
- 数据来源：各政府官网公开日程 + 人工维护

**② 关键词监控告警**
- 用户可订阅关键词（如"无人机"、"高企认定"）
- 当天内新内容匹配到关键词时，标记 `🔔`
- 可选：邮件/微信推送（高级功能，不强制）

## 六、GitHub Actions 自动更新策略

### 当前配置（复用）
- 触发频率：每 30 分钟
- 自动提交：`data/*.json`
- 公开示例 OPML：`feeds/follow.example.opml`

### 针对中通吉版的改动

| 改动项 | 说明 |
|--------|------|
| 优先级调整 | 政策申报类信源优先抓取，AI新闻类降权 |
| 新增信源 | 按第四章清单，逐步接入政府官网 RSS + Jina 兜底 |
| 数据结构 | `data/latest.json` 需增加 `region` 字段（`region: "qingpu\|shanghai\|national"`）和 `funding_amount` 字段（如"最高补贴 500 万元"） |
| 申报日历 | 新增 `data/calendar.json`，GitHub Actions **每日**更新（不是每周一），Banner 从 `calendar.json` 读取，无数据时显示"暂无申报窗口期信息" |
| 部署分支 | `gh-pages` 分支，托管 `index.html` + `data/` |

### ⚠️ Jina 兜底抓取已知风险

| 风险 | 说明 | 应对 |
|------|------|------|
| 反爬/CAPTCHA | gov.cn 子站大量部署人机验证 | RSS 优先；Jina 失败则标记"人工监控"，不阻塞其他信源 |
| 动态渲染 | Vue/React 页面 Jina 可能抓到空壳 | 逐站验证；失败则换用 RSS 或标注为手动关注 |
| robots.txt 限制 | gov.cn 通常禁止爬虫 | 仅抓取公开页面；不绕过验证 |
| Jina API 限流 | 免费 tier 有 QPS 上限 | 控制抓取频率；30分钟全量抓取不现实，改为按需/每日 |
| 编码乱码 | 部分页面编码不规范 | 抓取后校验字段完整性；异常则丢弃 |

**降级策略**：RSS > Jina 验证通过 > 标注"人工关注" > 放弃

### GitHub Pages 部署

- 域名：`q1387154-spec.github.io/ai-news-radar`（或自定义域名）
- HTTPS：自动启用
- 无需服务器，完全免费
