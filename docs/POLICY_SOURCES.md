## 六、监控优先级排序

### P0 每日必扫（GitHub Actions 每6小时）

1. 上海经信委（数字化转型专项资金）
2. 上海发改委（重点项目）
3. 上海科委（高企/科小）
4. 青浦经委/科委（区县专项）
5. 工信部（产业政策）
6. 国家邮政局（快递监管）
7. 交通运输部（物流政策）

### P1 每日扫（GitHub Actions 每日）

1. 长三角示范区政策
2. 财政部/税务总局（税收优惠）
3. 商务部（商贸政策）
4. 人社部/人社局（稳岗补贴/人才）

### P2 每周扫（手动或周cron）

1. 揭榜挂帅清单
2. 政府采购意向
3. 产业基金动态
4. 预算草案
