# AI News Radar — 伯乐Skill / Scout Skill

## 核心定位

> **从一堆信源里选出千里马。**

这不是 RSS 大全，也不是新闻聚合器。
而是一个**产业政策情报网络**，帮助物流/快递企业在海量信息中找出真正值得行动的机会。

---

## 第一原则：三层情报架构

所有信息源必须归入以下三层：

### 第一层（最高可信）— 官方政策源
**作用**：决定能不能做、有没有钱

| 来源 | 优先级 | 类型 |
|------|--------|------|
| 国家发改委 / 工信部 / 科技部 / 财政部 | P0 | RSS |
| 上海市经信委 / 发改委 / 科委 | P0 | RSS |
| 青浦区经委 / 科委 / 财政局 | P0 | RSS |
| 国家邮政局 / 交通运输部 / 民航局 | P0 | RSS |

**判断标准**：官方文件决定申报资格，是所有判断的基准线。

---

### 第二层（高价值）— 行业解读源
**作用**：提前预判风向，识别政策起风方向

这是**最被低估的层次**。很多有价值的信息：
- 官方文件很晚才出
- 行业解读很早就有了

**来源类型**：
- 产业研究公众号（RSS 桥接）
- 政策解读类博客
- 智库研报
- 行业垂直媒体

**判断标准**：能提前识别政策方向的信号，而不只是转发文件。

---

### 第三层（机会层）— 企业案例 / 地方试点
**作用**：谁会先拿到钱

- 地方试点公示
- 中标公告
- 企业项目公示
- 专项资金分配名单

**判断标准**：能看到谁在动作，比官方文件更接近钱。

---

## 第二原则：传播链分析

真正的政策传播顺序经常是：

```
智库吹风
    ↓
行业公众号（提前解读）
    ↓
地方试点
    ↓
部委征求意见稿
    ↓
正式政策
```

**伯乐的任务**：不追最末端的正式文件，而是顺着传播链往上追。

当一个行业公众号**连续多次**命中关键词，就说明这个源值得加入「高价值源池」。

---

## 第三原则：公众号价值评分

不做"公众号大全"。对每个来源打**源评分**：

```
Source Score = 政策提前量 × 30% + 专业度 × 30% + 原创率 × 20% + 产业关联度 × 20%
```

| 维度 | 说明 |
|------|------|
| 政策提前量 | 多次在官方文件前出现相关关键词 |
| 专业度 | 内容深度，不是泛泛而谈 |
| 原创率 | 有实地调研、数据、行业资源 |
| 产业关联度 | 与物流/快递/AI直接相关 |

**自动升级规则**：
- 某公众号连续 5 次命中某关键词 → 自动进入「高价值源池」
- 某源连续 30 天无高质量命中 → 自动移出

## 第四原则：关键词赛道分类

每个赛道维护独立的关键词列表和订阅源：

### 赛道 1：低空经济 / 无人配送

```text
低空经济, eVTOL, 无人机物流, 空域改革, 低空物流,
无人配送车, 自动驾驶配送, 车路云, 末端配送
```

**高价值源**：低空经济观察, 无人机之家, 亿欧汽车, 赛迪顾问, 甲子光年, 物流指闻, 新战略低空经济研究院, 智慧交通

### 赛道 2：超长期国债 / 两新

专项债, 中央预算内投资, 现代物流, 仓储冷链

**高价值源**：投实, 产业政策观察, 前瞻产业研究院, 零壹智库, 智能制造之家, 工业互联网世界

### 赛道 4：专精特新 / 高企 / 申报

## 第五原则：文章反向锁定公众号

**不订阅公众号，而是订阅「关键词命中后的公众号」**。

流程：
1. 某篇文章命中关键词（通过 RSS / 搜索）
2. 从文章溯源到公众号
3. 判断该公众号是否值得加入源池
4. 如果连续命中 → 自动升级为固定追踪源

**不是列表很长，而是命中质量很高**。

---

## 添加来源判断流程

当用户给出一个新来源，按以下顺序判断：

```
1. 是官方RSS吗？ → 直接加第一层
2. 是行业公众号/博客吗？ → 进入源评分流程
3. 能抓到结构化数据吗？ → 判断是否值得解析
4. 只是蹭关键词吗？ → 跳过或降级
5. 连续命中关键词？ → 加入高价值源池
```

**跳过的标准**：
- 纯转发，无原创内容
- 粉丝数高但内容同质化
- 与物流/AI/政策无直接关联
